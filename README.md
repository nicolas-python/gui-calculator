# Grafischer Taschenrechner 

Ein Grafischer Taschenrechner in Python für die Grundrechenarten.
Grafische Benutzeroberfläche mit Tkinter erstellt.

## Funktionen
- Addition
- Subtraktion
- Multiplikation
- Division
- Berechnungen
- Löschen vom EIngabefeld

## Nutzung 
1. Programm starten (python calculator.py).
2. Die gewünschte Rechenart auswählen, indem man die entsprechenden Buttons klickt.
3. Zwei Zahlen eingeben.
4. Auf = klicken, um das Ergebnis zu erhalten.
5. Mit C das Eingabefeld löschen, um eine neue Berechnung zu starten.
